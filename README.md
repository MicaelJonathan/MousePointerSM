# MousePointerSM

PORTUGUÊS ----
Software feito para "Simular" sons ao click e arrasto do mouse. Feito em C e C++ usando somente API do windows para
menor latência possivel e funcionar na maior quantidades de hardwares possiveis.

O programa aceita .wav e .riff mono 44100hz em 16 bit. Os samples estão localizados na pasta "samples" dentro do release, mas
se feito corretamente qualquer arquivo de som pode ser usado. 

Para build do projeto importe o mesmo em alguma IDE de sua escolha (o programa foi feito no Visual Studio) e use o compilador g++.

ENGLISH ---- 
Software made to "Simulate" sounds on mouse clicks and drag. Coded in C and C++ using only windows API for least possible latency and to work on as many hardwares as possible.

This software accepts .wav and .riff mono, 44100hz in 16 bit. All the samples are locaded on the folder "samples" inside the release, but if done correctly any sound can be used.

To build the project import the source code to any IDE of your choice (this project was done using Visual Studio) and use the g++ compiler.
